# feature/question-data Plan

## Goal

앱이 소비할 정적 문제 데이터를 준비한다.

## Scope

- `questions.json` 작성
- 샘플 문제 데이터 생성
- 데이터 로딩 방식 결정

## Out of Scope

- 문제풀이 UI
- 채점 로직

## Deliverables

- `data/questions.json`
- 필요 시 데이터 로더 유틸

## Validation

- JSON 구조가 타입과 일치하는지
- `id`와 `answer` 값이 유효한지
